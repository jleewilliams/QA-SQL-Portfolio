# Josh Williams \| QA SQL Portfolio

Assurance and data validation scenarios. The projects are framed as
real-world QA investigations and reporting validations rather than
academic SQL exercises.

-   Reporting Validation
-   Data Integrity Testing
-   Business Rule Validation

-   Defect Investigation
-   Regression Testing Support

## Featured Portfolio Projects

### QA-SQL-001: User Login Frequency Distribution

Validates user activity reporting and ensures inactive users are
included in dashboard calculations.

### QA-SQL-003: Employee Relationship Data Validation

Tests data integrity across employee, manager, department, and project
tables.

### QA-SQL-004: Top 3 Baby Names by Gender

Validates ranking logic using ROW_NUMBER() and PARTITION BY.

### QA-SQL-005: Sales Running Total and Running Average

Verifies cumulative KPI calculations using window functions.

### QA-SQL-006: Daily Sales Previous-Day and Next-Day Comparison

Uses LEAD() and LAG() to validate day-over-day trend reporting.

### QA-SQL-009: Quarterly Video View Validation

Validates quarterly KPI calculations and dashboard metrics.
